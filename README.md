## Description
This project is a Python web scraping script that extracts quotes, authors,
and tags from a website and saves them into a CSV file.

- Saves data into CSV format
- Clean and beginner-friendly code
- Uses popular Python libraries

## Technologies Used
- Python 3
- requests
- BeautifulSoup
- csv

## How to Run
1. Install dependencies:
   pip install requests beautifulsoup4
2. Run:
   python scraper.py
